Architectura

Architectura is a professional, responsive static website built using HTML, CSS, and Bootstrap Grid, enhanced with WOW.js, Odometer.js, and Lenis.js for smooth animations and interactive scrolling. It features a clean architectural layout with modern UI elements, animated counters, and dynamic effects for an engaging user experience. Optimized for performance and mobile devices, the project demonstrates strong skills in front-end development, responsive design, and the integration of third-party animation libraries to create a polished, portfolio-ready interface.

Tesla Clone

The Tesla HTML Clone is a responsive and visually appealing static website built using HTML, CSS, and basic JavaScript. Inspired by Tesla’s official homepage, the project replicates key design elements such as clean layout, smooth scrolling, and modern UI aesthetics. This clone highlights my skills in structuring web pages, creating intuitive navigation, and applying responsive design principles for a seamless user experience across devices. Goblin Hunt

This is a 2D side-scrolling shooting game built with Python and Pygame, featuring animated character movement, jumping, shooting, and enemy interactions. Players control a character to eliminate enemies while avoiding collisions. The game includes collision detection, a scoring system, background music, and sound effects. When the score reaches zero, a “Game Over” screen is displayed. The project demonstrates skills in game loops, OOP, event handling, and multimedia integration using Pygame.
